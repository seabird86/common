# Getting Started

## For Develop

1. Push the jar file up to a `Local Repository`. Open Maven Dialog -> {projectName} -> Plugins
 - clean -> clean:clean
 - compiler -> compiler:compile
 - jar -> jar:jar
 - install -> install:install-file


## For Release


### Devtools

```
-Dspring.devtools.restart.enabled=false
```
